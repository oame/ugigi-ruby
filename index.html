Ugigi

Ugigi Wrapper for Ruby 1.9.x

Requirements

  • Ruby 1.9.x
  • mechanize gem

Installation

gem install ugigi

Usage

require "ugigi"

# フリーワード検索で"ナズーリン"が含まれているSSを持ってくる
naz_novels = Ugigi.search(:free => "ナズーリン")
puts naz_novels.size

# "寅丸星"がタグに設定されているSSを持ってくる
novels = Ugigi.search(:tag => "寅丸星")
novels.each do |novel|
    puts novel.title
end
